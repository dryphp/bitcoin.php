bitcoin-php
===========

A [Bitcoin][Bitcoin] library for [PHP](http://www.php.net/).

Documentation
-------------

Documentation can be found at [code.gogulski.com](http://code.gogulski.com/).

Requirements
------------

### PHP requirements:
* PHP5
* cURL support  
* SSL support (if you're using HTTPS to talk to bitcoind)
